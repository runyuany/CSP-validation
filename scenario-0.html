Scenario 0: No CSP Header

CSP Header Used:

None (No Content-Security-Policy header is set)

Test Result:

The box below should have a light green background if external CSS is loaded successfully.

Otherwise, it will remain gray (indicating the CSS was blocked by CSP).

This is the test element. If you see a light green background, the external CSS was loaded successfully.

Expected Outcome:

CSS loads. Box is green. No CSP violations.

How to Verify:

  1. Check the appearance of the box above - is it green (CSS loaded) or gray (CSS blocked)?
  2. Open your browser's developer tools (F12) and look at the Console tab for any CSP violation messages.
